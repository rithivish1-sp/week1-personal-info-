# Personal Information Manager 
## Project Description 
This is my first Python project! It's a program that stores and displays personal information.
## What I Learned 
1. **Variables**: How to store different types of data
2.  2. **Input/Output**: Getting user input and displaying results
    3.  3. **String Formatting**: Using f-strings to create nice output
        4. 4. **Error Handling**: Basic validation for user input
 ## How to Run This Program 
 1. Make sure you have Python installed
 2.  2. Open terminal/command prompt
     3. 3. Navigate to the project folder
        4. 4. Run: python personal_info.py
           5. 5. Follow the prompts to enter your information
## Features 
Stores static information (name, age, city, hobby) - 
Gets dynamic information from user (favorite food, color) -
Displays all information in a formatted way - 
Basic input validation - Age calculation in months
